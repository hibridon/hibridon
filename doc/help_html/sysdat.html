Subroutine Sysdat

The subroutine SYSDAT controls the I/O of all system dependent parameters as well as the parameters NTERM, LAMMIN, LAMMAX, and MPROJ.

Currently the HibridonTM package contains SYSDAT subroutines for different collision systems described in more detail in the Systems help file.

See also the BASIS help file.
