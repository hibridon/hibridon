Energies

The values of the collision energy at which the scattering calcuation will be run are specified by the input line

    ENERG = {energy1},{energy2}...{energyN}\

If other input data occurs on the same line, then the string of energies must be terminated by a backslash.

Note that if the length of the array of energies is greater than the current value of NERG, then NERG is reset accordingly. The maximum number of energies cannot exceed 10.
