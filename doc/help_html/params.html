Parameters

The scattering calculation is controlled by GENERAL (system independent) and SYSTEM parameters (those which are specific to a particular collision system). More information on the collision systems which can be treated is given here.

The system independent parameters are:


ENERG | FSTFAC | INDOUT | IPRINT | JOUT | JTOT1 | JTOT2 |
JTOTD | JLPAR | LSCREEN | NERG | NUMAX | NUMIN | NUD
| RCUT | RENDAI | RENDLD | RINCR | RSTART | SPAC | TOLAI
| XMU
To change values of any parameters enter the command

PARAMETER1=VAL1,PARAMETER2=VAL2 etc.

Different parameters can be separated by commas or by semicolons

Note that values for the parameters ENERG, INDOUT, or JOUT must be terminated by a backslash (\) if other PARAMETER=VAL strings follow, e.g.

ENERG=1000,2000\AIRYFL=T

The HibridonTM input parser is not case sensitive, so parameters can be entered in either upper or lower case. Parameters may be abbreviated, although you will be asked to re-enter the parameter if there is an ambiguity.

Go for an example of how to enter and/or change parameters.

The default parameters are those for the collision of N2 with Ar, which is used a test case.
