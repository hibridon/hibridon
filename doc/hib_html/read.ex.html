Examples of the commands READ, SAVE, SHOW, and QUIT

In this example, I have added illustrative comments in brown to the actual output of the run. In addition, the various commands have been linked to the corresponding help files

The output which follows corresponds to an earlier version (4.0) of the code. However, the procedures illustrated here are unchanged in version 4.3

 Hibridon> rc=20

 Hibridon> wr=t

 Hibridon> wrp=t
