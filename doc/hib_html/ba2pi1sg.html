Basis Subroutine and System Specific Parameters: Collisions between a 2Π Molecule and a 1Σ Molecule

BASISTYPE = 20

System subroutine: sy2pi1sg

Basis subroutine: hiba2pi1sg.f

A 2Π Molecule and a 1Σ Molecule
Ref. S. M. Miller and D. C. Clary, J. Chem. Phys. 98, 1843 (1993); A. R. Offer and M. C. van Hemert, J. Chem. Phys. 99, 3836 (1993); P. E. S. Wormer et al. J. Chem. Phys. 122, 244325 (2005).

Unlike earlier basis routines, this basis routine allows the user to define which terms to include in the angular expansion without limitations other than the total number of terms (which is fixed in Hibridon). This information must be written to the lms array in the mod_ba2pi1sg module by the loapot subroutine. See pot_ohh2.f for an example.

Coupled states calculations are NOT supported in this basis routine.

The definition of the system dependent parameters is as follows:

J1MAX: the maximum rotational quantum number, minus 1/2, in the channel expansion for the 2Π molecule.

J1MAX is an integer variable. Because we are dealing here with a molecule with half-integer spin, the allowed channel rotational angular momenta run from 1/2 to JMAX+1/2.

NPAR: number of lambda-doublet components included.

If NPAR = 1, just ε = +1 states are included.
If NPAR .ne. 1, both ε = +1 and ε = - 1 states are included.

J2MIN: the minimum rotational angular momentum for the 1Σ molecule.

J2MAX: the maximum rotational angular momentum for the 1Σ molecule.

IPOTSY2: step used in iterating the rotational angular momentum for the 1Σ molecule.

For H2, IPOTSY2 can be set to 2 so that either para (even J2MIN and J2MAX) or ortho (odd J2MIN and J2MAX) levels are included.

BROT: rotational constant of the 2Π molecule

ASO: spin-orbit constant of the 2Π molecule

P, Q: lambda-doubling constants of the 2Π molecule

DROT: rotational constant of the 1Σ molecule
