Basis Subroutine and System Specific Parameters: Collision Involving the 1D and 3P States of an Atom with Electron Configuration p2 or p4 of an Atom with a Structureless Atom

BASISTYPE = 22

    System subroutine: sy1d3p

    Basis subroutine: hiba1d3pp.f

Ref. P. J. Dagdigian, M. H. Alexander, and J. Klos J. Chem. Phys. 143, 054306 (2015).

The definition of the system dependent parameters is as follows:
    NTERM: the number of potential surfaces involved. This parameter can not be changed

    NSTATE: number of electronic states included:

      set NSTATE=0 for just the 1D state
      set NSTATE=1 for just the 3P state
      set NSTATE=2 for both 1D and 3P states

    The definition of the real system dependent parameters is as follows:

      EN1D: energy of the 1D state relative to that of ths 3P state.
